# desafio-final-python-ia-aplicada
Desafiio final do curso de Python: Inteligência Artificial Aplicada

Objetivos:

1 - Carregar um arquivo .txt, onde cada linha será um elemento da lista Python

2 - Mandá-la ao modelo rodando localmente para extrair, em formato JSON, onde cada item terá "usuario", "resenha_original", "resenha_pt", "avaliacao" (Positiva, Negativa ou Neutra)

3 - Transformar a resposta do modelo em uma lista de dicionários Python

4 - Criar uma função que, dada uma lista de dicionários, percorre a lista e faz 2 coisas:

  a ) Conta a quantidade de avaliações positivas, negativas e neutras
  
  b ) une cada item dessa lista em uma variável do tipo string com algum separador
  
  c ) ao final, retorna ambas as coisas
